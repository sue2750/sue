# 私密相册使用说明

## 如何清除所有照片

我已经为您创建了一个专门的页面来清除所有照片数据：

### 方法一：使用清除页面（推荐）
1. 打开 `clear-all-photos.html` 文件
2. 点击"确认清除所有照片"按钮
3. 在弹出的确认框中再次确认
4. 系统会自动清除所有照片并返回主页面

### 方法二：手动清除（适用于高级用户）
1. 在浏览器中打开开发者工具（通常按 F12 或 Ctrl+Shift+I）
2. 切换到 "Console"（控制台）选项卡
3. 输入以下命令并按回车：
   ```javascript
   localStorage.removeItem('galleryPhotos');
   location.reload();
   ```
4. 刷新页面即可看到所有照片已被清除

- **导航**：使用上一张/下一张按钮或键盘方向键浏览照片

## 注意事项

- 所有照片数据都存储在您的浏览器本地存储中
- 清除照片后数据无法恢复，请谨慎操作
- 支持的图片格式：JPG、PNG、GIF
